# Proyecto-Tareas-Gestor
## Introduccion
* Este es un proyecto de pagina web que permite organizar tareas en un kanban mediante un formulario.


## Funcionalidades
* Crear tareas
* Mover y asignar importancia.
* Actualizar estados de tareas por proceso
* Eliminar tareas

## Herramientas usadas
- ChatGPT
- JavaScript
- Html
- Css
